1

Upload Your Documents

Start by uploading your documents in various formats. The system supports:

  • PDF documents (including scanned with OCR)
  • Microsoft Word documents (.docx)
  • Text files (.txt, .md, .csv)
  • ZIP archives (automatic extraction)

Tip: You can upload multiple files at once or entire ZIP archives containing multiple documents.

2

Build Your Knowledge Base

Click "Start Building" to process your documents through the complete pipeline:

  • Parsing: Extract text and metadata from all documents
  • Chunking: Split content into optimal chunks using adaptive strategies
  • Embedding: Generate vector embeddings for semantic search
  • Indexing: Build vector store and keyword search indices

Tip: Monitor the real-time progress bar to track each processing step.

3

Chat with Your Documents

Once processing is complete, start asking questions in natural language:

  • Ask factual questions about your content
  • Request summaries of specific topics
  • Compare information across multiple documents
  • Get analytical insights from your data

Tip: Each response includes source citations so you can verify the information.

4

Analyze and Optimize

Use the analytics dashboard to monitor performance and quality:

  • View Ragas evaluation metrics for response quality
  • Adjust configuration parameters in real-time
  • Export conversation history for analysis
  • Fine-tune retrieval and generation settings

Tip: Experiment with different chunking strategies and temperature settings to optimize for your specific use case.
